# BRICO

Brico is composed to:

* A webapp
* A mobile application

## Pre-requist

Docker and docker-compose are needed to start common architecture elements

## Start the application

Use docker compose

```sh
docker compose ud -d
```

### Webapp

Webapp code is in [webapp](./webapp) folder, follow the [README.md](./webapp/README.md) to start the application

### Mobile application

Mobile application code is in [mobileapp](./mobileapp) folder, follow the [README.md](./mobile/README.md) to start the application
